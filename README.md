🎬 Movie Review Website (Trial Project)
📌 Overview

This is a trial movie review website built to explore basic web development concepts such as UI design, content structuring, and user interaction. The platform allows users to browse movies and read or write reviews in a simple and clean interface.

🚀 Features

🎥 Browse a list of movies

✍️ Add and read reviews

⭐ Rating system for movies

🔍 Simple and intuitive UI

📱 Responsive design (if applicable)

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: (Node.js)
